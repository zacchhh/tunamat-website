Features
Everything you need to plan
the perfect class

From a 272+ movement library to live class execution — TunaMat gives pilates instructors the tools to create balanced, beautifully paced classes.

📚

272+ Movement Library

A curated library of mat pilates movements with rich metadata — muscle groups, movement patterns, equipment, and pacing presets. Continuously refined with input from professionals.

🧩

Drag & Drop Flow Builder

Create flows with segments like Warm Up, Core Work, and Cool Down. Add movements with individual pacing and instructor notes.

⏱

Timed Play Mode

Execute your flow live in class with countdown timers per segment. Designed to be glanceable while you teach.

🔥

Muscle Group Analysis

See a heatmap breakdown of which muscles your flow targets. Catch imbalances before class, not after.

🎵

Spotify Integration

Import any public Spotify playlist and map songs to segments. Reference BPM and duration for perfect pacing.

☁️

Cloud Sync & Offline

Your flows sync across devices via the cloud. Works offline too — changes sync when you reconnect.
