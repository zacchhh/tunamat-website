Core Tuna Features

Every app in the Tuna Athletics suite shares a common set of powerful features — designed with real instructors, refined by the community.

🎵

Music-First Planning

Build your class around the music. Map any playlist to your segments with BPM data for perfectly paced transitions — designed around how instructors actually plan.

📚

Professional Movement Library

Hundreds of exercises vetted by industry professionals, with rich metadata — muscle groups, movement patterns, equipment, and recommended pacing.

👥

Community-Driven

Movement libraries are maintained and improved by the user community. Your expertise shapes the platform — create custom exercises and share knowledge.

⚡

Rapid Class Planning

An intuitive interface that speeds up class planning from hours to minutes. Drag-and-drop segments, add movements, and set pacing in seconds.

⏱

Live Play Mode

Hit play and teach. Glanceable timers guide you through each segment with crossfade-aware pacing — the ultimate in-class companion.

📊

Real-Time Analytics

Muscle group analysis and balance feedback give you real-time insight pre-execution. Catch imbalances before class, not after.
